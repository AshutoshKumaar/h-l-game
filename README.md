# Higher Lower Game

A Next.js and Tailwind CSS higher-lower game with split-screen image panels, hover reveals, local high score, and a localStorage leaderboard.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Notes

- Game rounds are fetched from `https://beautyglamours-f0ec7.web.app`.
- Scores and leaderboard entries are stored in browser localStorage.
